Speckle's "Beyond interop" session presents design data as a portfolio-scale resource rather than a file exchange problem. The session covers enterprise data layers, reducing rework, improving delivery confidence, and using design intelligence workflows across projects.

The update also highlights recent Speckle capabilities including a refreshed UI, project archiving for future reference and analysis, and the general availability of model data validation workflows.
