You can join the OSArch live chat from Matrix or IRC. Both connect to the same community conversation.

## Matrix

Matrix is the easiest option if you are new to live chat.

<a class="button" href="https://app.element.io/#/room/#OSArch:matrix.org">Join OSArch on Matrix</a>

1. Go to [app.element.io](https://app.element.io/).
2. Create an account or sign in.
3. Choose “Explore rooms” or “Join room”.
4. Enter this room address:

   `#OSArch:matrix.org`

5. Join the room and say hello.

You can also use any other Matrix app. The room address is always:

`#OSArch:matrix.org`

## IRC

IRC is an older chat system used by many open source communities.

Server:

`irc.libera.chat`

Channel:

`#OSArch`

Beginner-friendly IRC options include:

- [web.libera.chat](https://web.libera.chat/)
- HexChat
- WeeChat
- irssi

To join from the Libera web chat:

1. Open [web.libera.chat](https://web.libera.chat/).
2. Pick a nickname.
3. Enter `#OSArch` as the channel.
4. Connect and say hello.
