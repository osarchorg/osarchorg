[Bonsai v0.8.3](https://community.osarch.org/discussion/comment/25777/#Comment_25777) has been released with 770 new features and fixes. The release notes describe this as a polish-heavy update, with a lot of work going into bug fixes, usability, optimisation, and smaller improvements across the project.

Headline changes include Blender 4.5 support, an overhaul of bSDD support with multiple dictionaries and improved dictionary navigation, IFC5D improvements, array support for openings and spatial elements, editable topology representation items, Python 3.9 support across utilities, a more robust IfcSQL, and interface tools for managing `IfcApplication`, organisations, people, and applications.

The release also lands alongside growing training and community material, including Ryan Schultz's Bonsai tutorial library, Yassine Oualid's AECO.DEV IfcOpenShell programming course, and the launch work around BIMVoice Academy. Download Bonsai from [bonsaibim.org](https://bonsaibim.org/) and read the [full release notes](https://community.osarch.org/discussion/comment/25777/#Comment_25777).
